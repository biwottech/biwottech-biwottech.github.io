## ğŸš€ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/biwottech/portfolio.git
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser

### Building for Production

```bash
npm run build
```

### Deploying to GitHub Pages

```bash
npm run deploy
```

## ğŸ“ Customization

### Adding Your Content

1. **Personal Information**: Update the content in each section component
2. **Projects**: Modify the projects array in `src/sections/Projects.tsx`
3. **Experience**: Update the experiences array in `src/sections/Experience.tsx`
4. **Skills**: Modify the skills array in `src/sections/About.tsx`
5. **Contact Info**: Update contact methods in `src/sections/Contact.tsx`

### Adding Project Images

1. Add project screenshots to `public/projects/`
2. Use these filenames:
   - `me-system.jpg`
   - `rbm-system.jpg`
   - `pharmacy-system.jpg`
   - `hotel-system.jpg`
   - `campaign-system.jpg`
   - `ecommerce-system.jpg`

### Adding Your CV

1. Replace `public/Evans_Kibiwott_CV.pdf` with your actual CV file
2. Keep the same filename or update the references in the code

### Customizing Colors

The portfolio uses Tailwind CSS with a primary color scheme of indigo. To change colors:

1. Update color classes throughout the components
2. Modify the gradient colors in the Hero section
3. Update the color scheme in `tailwind.config.js` if needed
